# Weather
Консольная программа на Go для получения текущей погоды в городе с использованием API Яндекс.Погоды. Проект создан в учебных целях для изучения Go, работы с HTTP, JSON.

## Технологии
- [Go 1.22.2](https://go.dev/)
- [godotenv](https://github.com/joho/godotenv)

### Требования
- Установить [Go](https://go.dev/dl/) версии 1.22+
- API-ключ Яндекс.Погоды (бесплатный для "Умного дома")

### Начало работы

1. Клонируйте репозиторий:
```bash
git clone https://github.com/твой_аккаунт/Weather.git
cd Weather
```

2. Установить зависимости
```bash
go mod tidy
```

3. Создайте файл `.env` в корне проекта (используя `.env.example` как шаблон) и вставьте туда свой API-ключ.
(Его можно получить зарегистрировавшись в `Яндекс погода для умного дома`. Бесплатно.)

4. Запустите программу:
```bash
go run cmd/cli/main.go Москва
```
